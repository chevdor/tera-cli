# Install

## From source

```
cargo install --git https://github.com/chevdor/tera-cli
```

## Linux
```
wget {{ DEBIAN_URL }}
dpkg -i tera-cli_linux_amd64.deb
tera --help
```

## MacOS

```
brew tap chevdor/tera-cli https://github.com/chevdor/tera-cli
brew update
brew install tera-cli
```
